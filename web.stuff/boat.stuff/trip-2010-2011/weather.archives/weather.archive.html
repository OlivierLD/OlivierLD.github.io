This page is gathering the weather archives we used during the trip. They are made of faxes and GRIBs. They can be rendered by the soft we've developped for the occasion, available for free on its web page.
You can do several things:

  • Download the archive (.waz extension) and open it with the program mentioned above.
  • Copy the URL of the archive (.waz extension) and open it with the program mentioned above. File > Load Composite from URL

The program allows you to show or hide any layer (faxes, gribs, chart, boat...)
