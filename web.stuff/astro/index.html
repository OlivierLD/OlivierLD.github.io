  • ES6
    The links below are illustrations of what can be done with the ES6 part of this git repo, which remains the point of truth.
    The "backend" part - for the calculations - is available in several languages (Java, C, ES6, PHP, Python, Scala, Go).
    The WebComponents are defined in their own git repo.

    • Dynamic Globe Background (with WebComponents)

      This shows a rich Web UI, that can be served by a small machine (like a Raspberry pi Zero). The small machine provides the data, the data UI is rendered on another device, connected on the same network as the small machine (possibly emitted by it).
      The UI rendering is the most demanding process, taken care of by a phone, tablet, or laptop, connected on the network, and not by the server itself.

      This brings us back to this Entropy Zero (EZ) computing paradigm. Network, but no Internet. Not cloud. Flake Computing.
