    • Dynamic Globe Background (with WebComponents)
      This shows a rich Web UI, that can be served by a small machine (like a Raspberry pi Zero). The small machine provides the data, the data are rendered on another device, connected on the same network as the small machine (possibly emitted by it).
      The UI rendering is the most demanding process, taken care of by a phone, tablet, or laptop, connected on the network.
