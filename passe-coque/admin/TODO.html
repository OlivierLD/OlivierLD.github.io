MySQL & PHP

MySQL est un moteur de bases relationnelles, disponible (par défaut, sans supplément) chez notre hébergeur.
Le sigle SQL signifie Structured Query Language. C'est un langage (de programmation) destiné à interagir avec (et pas seulement interroger) les bases de données relationnelles.
PHP est un langage utilisé pour générer du HTML dynamique, et qui est aussi équipé pour accéder à une base MySQL.
Le sigle PHP peut signifier Personal Home Page, ou Popular Hypertext Processor..., c'est vague.

Afin de savoir comment élaborer le schéma de la base de données (structure des tables et des contraintes), il s'agit d'abord de savoir ce qu'on en attend.
Par exemple :

  • On veut pouvoir faire une rappel aux membres dont la cotisation expire dans moins d'un mois
  • On veut la liste des membres qui ne sont pas abonnés à la news letter, et vice-versa
  • On veut que chaque membre ait un espace privé (protégé par un mot de passe) pour y voir ses cotisations, son statut, etc.
  • On veut un espace d'administration (protégé par un mot de passe) pour la publication des news letters, administration de la base de données (avec une interface web), etc.
